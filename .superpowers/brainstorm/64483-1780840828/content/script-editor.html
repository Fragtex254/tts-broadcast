页面行为

✓ 下半区左侧：VoiceGenerator（音色选择 + 切分生成按钮）
✓ 下半区右侧：SegmentEditor + AudioPlayer 上下堆叠
✓ 入口 1：从信源收集改写后自动跳转，store 中 script 和 currentBroadcast 已就绪
